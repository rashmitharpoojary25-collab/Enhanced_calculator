# Enhanced Calculator

A console-based calculator built in Java using OOP principles.

## Features
- Basic Arithmetic (Addition, Subtraction, Multiplication, Division)
- Scientific Calculations (Square Root, Power)
- Unit Conversions (Temperature, Currency)
- BigDecimal precision for accurate calculations
- Exception handling for invalid inputs

## Project Structure
src/
└── com.calculator/
├── basic/BasicCalculator.java
├── scientific/ScientificCalculator.java
├── converter/UnitConverter.java
└── main/Main.java

## How to Run
1. Install JDK 17 or above
2. Open project in IntelliJ IDEA
3. Run Main.java

## Technologies Used
- Java JDK 26
- BigDecimal for precision
- OOP - Inheritance, Encapsulation
